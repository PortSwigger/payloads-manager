Payloads Manager organizes and manages custom payloads within a dedicated tab interface, streamlining payload storage and retrieval during security testing. The extension provides comprehensive payload management with categorization, import/export functionality, and context menu integration for seamless workflow integration.

Features

  • Centralized payload storage with category organization
  • Add payloads directly from selected text in request/response editors
  • Quick payload insertion via right-click context menus
  • JSON import/export for payload sharing and backup

Usage

  1. Access the Payloads Manager tab to view and manage your payload collection
  2. Create categories using "Manage Category" to organize payloads by type or purpose
  3. Add payloads manually using "Add Payload" or right-click selected text in any request/response editor and choose "Add to Payloads Manager"
  4. Insert stored payloads by right-clicking in request/response editors and selecting from the "Payloads Manager" submenu organized by categories
  5. Use "Export as JSON" to backup payloads or "Import as JSON" to load payload collections
